<details>
<summary>表主题的助词</summary>

私　が　学生。

助词「は」和「が」其实很不一样。助词「が」用来识别某物或某人的属性，而助词「は」只是用来带出一个新的话题。

</details>


<details>
<summary>な形容词的活用规则跟名词是一样的。</summary>

第一张表：だ・だった・じゃない・じゃなかった

|        | 肯定   | 否定  |
|  ----  | ----  | ----  |
| 非过去  | 好きだ | 好きじゃない |
| 过去    | 好きだった | 好きじゃなかった |

第二张表(礼貌形式)：です・でした・じゃありません

|        | 肯定   | 否定  |
|  ----  | ----  | ----  |
| 非过去  | 好きです | 好きじゃありません |
| 过去    | 好きでした | 好きじゃありませんでした |

</details>

<details>
<summary>三种活用形来直接修饰名词（记住非过去肯定形后面要加「な」）。</summary>

魚が好きな人。
喜欢鱼的人。
魚が好きじゃない人。
不喜欢鱼的人。
魚が好きだった人。
以前喜欢鱼的人。
魚が好きじゃなかった人。
以前不喜欢鱼的人。

</details>

<details>
<summary>动词。</summary>

所有る动词以「る」结尾，而う动词则以包括「る」在内的多种う元音结尾。所以，一个不是以「る」结尾的动词肯定是う动词。对于以「る」结尾的动词，如果在「る」音之前的假名包含 /a/、/u/ 或者 /o/ 元音，那它就是う动词，而前面是 /i/ 或者 /e/ 元音的，那它大多数情况下就是る动词。一些常见的例外列在本节末尾。

</details>

<details>
<summary>动词否定</summary>

ある （う动词） - 存在（无生命）

いる （る动词） - 存在（有生命）

＊ある → ない

お金 が ない。

ねこ　は　いない。（没有猫。）

</details>


<details>
<summary>动词过去形</summary>

动词变成た form

</details>

<details>
<summary>动词过去否定</summary>

基本上所有的否定式都是以「ない」结尾。动词的过去否定式活用规则跟其他那些以「ない」结尾的否定式规则一样，只要把动词否定式结尾「ない」里的「い」去掉，换成「かった」就行。

</details>

<details>
<summary>直接对象的助词「を」; 目标助词「に」; 方向助词「へ」; 上下文助词「で」</summary>

助词「で」指明了动作发生的上下文。比如说一个人吃了鱼，那是在哪里吃的？如果一个人去了学校，他怎么去的？你喝汤配的什么？所有这些问题都可以用助词「で」来回答，如下例所示。

电影院で見た。

巴士で帰る。

餐馆で昼ご飯を食べた。

</details>
